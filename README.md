## MyFood

Ce projet est un site web de restaurant développé avec **Laravel**, **Tailwind CSS** et **Filament**.  
Il combine un **site vitrine moderne** destiné aux visiteurs et un **panel d’administration puissant** permettant de gérer facilement le contenu du site.

###  Fonctionnalités
- Présentation du restaurant (mise en avant des plats spéciaux, localisation, coordonnées)
- Affichage du menu
- Formulaire de contact avec validation et protection reCAPTCHA
- Envoi automatique d’e-mails à l’administrateur
- Panel d’administration sécurisé avec **Filament**
- Gestion du contenu (reserbation, menus, données du site) via le back-office
- Interface responsive et design moderne grâce à Tailwind CSS

### Technologies utilisées
- **Laravel** – Framework backend PHP
- **Tailwind CSS** – Framework CSS utilitaire
- **Blade** – Moteur de templates
- **Filament** – Panel d’administration pour Laravel
- **MySQL** – Base de données
- **SMTP** – Envoi d’e-mails

### Objectif
Fournir une solution complète pour un restaurant, avec un site web professionnel côté utilisateur et un espace d’administration flexible et personnalisable pour gérer le contenu facilement.
